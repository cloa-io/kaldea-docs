## Dashboard components

<img
  src="https://assets.cloa.io/documentation/dashboard/dashboard-components.png"

/>

A dashboard is comprised of 5 elements:

1. Title
2. Description
3. Labels
4. Filters
5. Content
   - Textbox
   - Page shortcut
   - Chart

## Creating a new Dashboard

There are a couple of ways to add a new dashboard in Kaldea.

1. Navigate to `Workspace` from the LNB. At the right of your `Public` and `Private` workspace, click `+` → `Add a dashboard`
2. Or, use the Cmd+K palette and type “Create new dashboard”.

Follow the guide below to get started on dashboards.
