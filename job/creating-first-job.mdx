# Job terminology

- **Refresh Query:** Refresh Query is a query that can update the table. Mostly
  INSERT with SELECT clause, or MERGE statements.

# Step 1 : Register a Refresh Query

<Note>
A refresh query can only be registered to tables created in Kaldea.

</Note>

## Registering Refresh Query using Query Editor

- To register a refresh query from the Query Editor, click `Save as` then toggle
  to the `Refresh query` tab from the Save as modal.
- Kaldea will automatically detect the datasource and destination table for you.
- Click `Save`.

![](/images/creating-first-job-2.gif)
